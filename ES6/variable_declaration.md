| Keyword | Scope          | Re-declaration & Re-assignment | Hoisting          |
|---------| -------------- |:------------------------------:| -----------------:|
| var     | Global, Local  | yes,yes                        | yes, with default |
| let     | Global, Local, block | no,yes                   | yes, without default | 
| const   | Global, Local, block | no,yes                   | yes, without default |
